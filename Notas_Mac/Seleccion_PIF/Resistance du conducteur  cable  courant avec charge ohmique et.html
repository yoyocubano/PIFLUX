Resistance du conducteur ( cable ) courant avec charge ohmique et mixte ; chute de tension et puissance dissipe


Resistance

R = ⍴ * L / A ou A = π*r^2 ou r= d/2

Courant

Ib = P/Un ( en courant continu et courant alterne charge ohmique ) ex resistance , ampoules , etc

Ib = P/Un * cosϕ ( courant alterne charge mixte ) moteurs monophasé

Ib = P /Un * √3( courant triphase avec charge ohmique )
Ib = P / Un * cosϕ*√3 ( courant triphase avec charge mixte )


Chute de tension

Δu= 2*Ib L⍴/ A ( en courant continu et courant alterne charge ohmique )

Δu= 2*Ib L⍴* cosϕ / A ( courant alterne charge mixte )


Δu= √3*Ib L⍴/ A ( courant triphase charge ohmique )

Δu= √3*Ib L⍴* cosϕ / A (courant triphase charge mixte )



Puissance dissipe

Pv = 2Ib^2 L⍴/ A

Pv = 3Ib^2 L⍴/ A


Notes


Etapes pour posse du câble :

  1. Determine Courant Ib ( plaque du moteur ) ou calcule
  2. Determine la type du posee et quantité du conducteur ( ex : pour cable sur mur dans un tube ; trois conducteurs charge pour triphasé )
  3. Determine la section du conducteur A ( type de pose B2 avec 3 conducteur charge pour Ir = 27 Amp ( Ib < Ir) A=4mm2
  4. Determine la courant nominal pour disjoncteur ( type de posse B2-3 dans le tableau 2 disjoncteur In = 25 Amp ; condition Ib <In <Ir)
